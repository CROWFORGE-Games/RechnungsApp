SMTP Hilfe

Google App-Passwort

So kommst du an das App-Passwort:

  1. In deinem Google-Konto die 2-Faktor-Anmeldung aktivieren.
  2. Danach App-Passwörter öffnen: myaccount.google.com/apppasswords
  3. Ein neues App-Passwort erzeugen.
  4. Den 16-stelligen Code ohne Leerzeichen in SMTP Passwort eintragen.
